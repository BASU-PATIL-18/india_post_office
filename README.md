This repository contains a comprehensive dataset of post offices across India, consolidating information from multiple regional datasets into a single unified table. The data is organized and merged on the basis of district names, ensuring a structured and easily navigable format. Post offices are categorized into three distinct types: Branch Offices (BO), which are smaller units serving rural and remote areas; Head Offices (HO), which function as the primary administrative centers managing postal operations within a region; and Sub Post Offices (SO/PO), which act as intermediaries between the Head Offices and Branch Offices, catering to semi-urban and urban localities.
The dataset was compiled by integrating multiple source tables, each representing postal data from different districts or administrative zones, and joining them under a common district identifier to form one consolidated master table. This makes the repository a valuable resource for analyzing the postal infrastructure of India, understanding the distribution of postal services across districts, and performing geographical or administrative studies related to India's postal network.
